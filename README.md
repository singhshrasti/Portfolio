# Blossom Dev Studio Portfolio

A personal portfolio website built with React, Vite, Tailwind CSS, and shadcn/ui components.

## Features

- Responsive portfolio layout with hero, about, experience, skills, projects, and contact sections
- Custom UI components using Radix UI and Tailwind CSS
- Client-side routing with React Router
- Modern build tooling with Vite and TypeScript

## Local development

```bash
npm install
npm run dev
```

Open `http://localhost:8080` in your browser.

## Build

```bash
npm run build
```

## Preview production build

```bash
npm run preview
```

## Deployment

This project is ready to deploy on Vercel.

### Vercel setup

1. Go to https://vercel.com/new
2. Connect your GitHub account
3. Select the `singhshrasti/Portfolio` repository
4. Use the default settings

Vercel will automatically run:

```bash
npm install
npm run build
```

The production site will be served from the root URL assigned by Vercel.

## Notes

- A `vercel.json` file is included to configure the static build and SPA routing.
- You can still keep GitHub Pages, but Vercel is now the recommended deployment path for this repo.
